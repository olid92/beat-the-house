# beat-the-house
Site for Project Beat the House (Season 2)

This repository contains the basic data sets for the decision making behind phase 2 of Project Beat the house. 

# source data
- result data for each season between 19/20 to 25/26, at match level
- dictionary for file schema

# transformed data
- Enhanced result data, collating seasons 19/20 to 25/26 in a single file
- Gameweek ranking data, simulating the league position of teams across each season, in a single file
